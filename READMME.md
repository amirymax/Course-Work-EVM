# Курсовой проект по численным методам

**Автор:** Зикрулло Амири  
**Год:** 2025  
**Язык реализации:** Python + PySide6

Проект представляет собой учебное приложение, объединяющее несколько классических численных методов в одном удобном графическом интерфейсе.  
С его помощью можно:

- численно интегрировать функции;
- решать нелинейные уравнения;
- строить интерполяционные полиномы;
- выполнять аппроксимацию методом наименьших квадратов (МНК);
- решать краевые задачи для ОДУ методом конечных разностей (МКР).

Все модули реализованы на Python, а интерфейсы — на **PySide6** с использованием **matplotlib** для построения графиков.

---

## Стек технологий

- **Python 3.10+**
- **PySide6** — создание GUI
- **NumPy** — работа с массивами и линейной алгеброй
- **Matplotlib** — визуализация графиков
